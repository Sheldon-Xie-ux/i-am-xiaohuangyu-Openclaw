# 🎁 OpenClaw 多 Agent Token 优化模板

## 这是什么？

一个开箱即用的 OpenClaw 多 Agent 配置模板，帮你：

- ✅ **降低 60% Token 消耗**
- ✅ **提升 2x 响应速度**  
- ✅ **实现 4x 任务并行**
- ✅ **5 分钟完成部署**

## 适用场景

- 有多个不同领域任务（交易+研究+内容）
- Token 消耗过高想优化成本
- 单 Agent 经常"神经错乱"
- 想让不同任务有专门专家处理

## 快速开始 (3步)

### 第 1 步: 复制模板

把这个文件夹发送给对方，对方解压后：

```bash
cd openclaw-multi-agent-template
```

### 第 2 步: 修改配置

1. 复制配置文件模板：
```bash
cp config/openclaw.json.template config/openclaw.json
```

2. 编辑 `config/openclaw.json`，填写：
   - **API Keys**: Kimi 或 OpenAI 的 Key
   - **群组 ID**: 飞书群设置里获取

### 第 3 步: 一键安装

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
openclaw gateway restart
```

## 工作原理

```
交易群消息 → trader Agent 处理 → 专业分析，Token 最优
研究群消息 → researcher Agent 处理 → 深度思考，不被干扰  
助手群消息 → assistant Agent 处理 → 快速响应，成本低
```

每个 Agent 有自己的记忆和上下文，互不干扰。
